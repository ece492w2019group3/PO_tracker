This library was generated using MATLAB C coder. It implements a Kalman filter to obtain orientation and angular velocity
from accelerometer, gyroscope, and magnetometer readings.
For more information on how this library works, please see the following link:
https://www.mathworks.com/help/fusion/ref/ahrsfilter-system-object.html

The contents of this folder include basic sample IMU data and plots of the changes in orientation along 3 axis for the IMU unit.
Much of the code for the orientation plots was taken from MATLAB's documentation for the Sensor Fusion and Tracking Toolbox, found here:
	https://www.mathworks.com/help/fusion/ref/ahrsfilter-system-object.html
The sample IMU data was obtained from the following website:
	https://www.nxp.com/pages/sample-data-sets-for-inertial-and-magnetic-sensors:SENSORDATA
